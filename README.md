## 💡 Featured Projects & Impact

- **AI-powered e-commerce / PIM platform** (PimPick) — **deep multi-provider AI integration** (OpenAI · Gemini · Claude · DeepSeek behind one abstraction layer). An **AI Wall** composes a wallpaper product into a customer's own room photo (image-to-image), plus AI catalog enrichment (descriptions/SEO/translation), AI vision (image → product data), and AI URL scraping — all async via a Node.js job queue, with an OpenCart 4 storefront and a Laravel + Vue 3/TypeScript console.
- **AI-powered verification SaaS** (InsureVerify) — **Google Gemini Vision** reads insurance/licence documents, extracts and name-matches the details, and feeds a composite risk score (IVRI) alongside **Checkr** background checks. Laravel + Vue 3/TypeScript with subscription billing, a token-authenticated developer API, and real-time chat.
- **Multi-tenant AI PIM SaaS** (platform-pim) — **four LLM providers** (OpenAI · Claude · Gemini · DeepSeek), AI vision, AI attribute/category mapping, an inline **AI Copilot**, and smart CSV import with AI column mapping. **React 19 + Express + Prisma/PostgreSQL** (87-model schema), Auth0 SSO.
- **Multi-tenant website-builder SaaS** (Uvisual, Node rewrite) — a drag-and-drop visual editor with **SSR multi-domain publishing**, reseller management, Mollie billing, and a legacy-PHP migration pipeline. **React 19 + TypeScript + Express + Prisma**, Docker/Caddy deployment.
- **Luxury car-rental platform + DevOps ownership** (Exotic Cars Rental) — a three-app system (**Next.js 16** storefront · **Vue 3** admin · **Laravel** API) with booking, availability, multi-currency pricing, OAuth, S3, and WhatsApp. I also **owned the CI/CD pipeline and VPS server setup** — GitHub Actions SSH deploys, PM2 zero-downtime reloads, on a **DigitalOcean Ubuntu 24.04** droplet (Nginx + Apache reverse proxy, UFW firewall, Certbot/Let's Encrypt TLS).
- **Multi-tenant B2B reseller SaaS** (FinditTrading & Compleetshop.cloud) — **19 live reseller storefronts**, each provisioned dynamically with its own isolated database and custom domain/subdomain. A **product-sync engine keeps ~14,000 products in sync** (options, categories, images, pricing, stock) from parent to every reseller, with orders forwarded back for cost-price and margin tracking.
- **Multi-vendor travel-booking marketplace** (Fyvco) — customized and extended a modular Laravel platform spanning **8 booking verticals** (hotels, tours, cars, flights, boats, events, spaces, airports), with vendor payouts, six payment gateways, and multi-currency / multi-language.
- **50+ projects delivered**, **14 Upwork projects at 5★ (Top-Rated)**, and a **2-developer team led** end-to-end through build, QA, and production support.
- **End-to-end Laravel document platform** (AugmentaInvoice) — quotation → proforma → invoice → packing list → purchase-order workflows, with Excel import/export, approvals, and configurable PDF export.
- **Mapping & geolocation for rental logistics** (WorkSecure) — extended a live Laravel invoice platform with unit-rental booking and a mapping layer: units and active bookings plotted on **Google Maps & Leaflet** with status-coded markers, address autocomplete + geocoding, and a **GPS feed tracking each unit's last-known location**.
- **Replaced a manual Photoshop workflow** (ProfichemApps) — an in-browser drag-and-drop label/poster editor with crop, resize, layering, and PDF/PNG export.
- **Custom OpenCart e-commerce** (BroditStore.nl) — a Dutch Brodit car-mount store with a **vehicle/device compatibility matrix** (brand · model · year), automated supplier feed sync with per-category pricing, bulk import/export, **Mollie** (15+ payment methods), and 39-language + GDPR support.
